# GEMINI TECH - 東方美學與 AI 整合組件庫

結合東方侘寂（Wabi-sabi）美學與次世代多模態 AI 能力的 Vue 3 專案。

## 專案特點

- **極簡主義**：採用 Naive UI，並透過 `all-in-one.css` 統一管理全域風格。
- **東方美學**：融入朱紅、墨黑、和紙白等經典色彩與禪意動畫。
- **AI 整合**：原生支援多模態 AI 介面展示，包含打字機效果與呼吸加載動畫。
- **純淨架構**：所有樣式均已整合至單一檔案，移除多餘樣式，保持開發直覺。

## 技術棧

- **框架**：Vue 3 (Composition API)
- **建構工具**：Vite
- **UI 庫**：Naive UI
- **語言**：TypeScript
- **樣式管理**：單一全域 `all-in-one.css`

## 開始使用

### 安裝依賴

```bash
npm install
```

### 本地開發

```bash
npm run dev
```

### 專案打包

```bash
npm run build
```

## 樣式說明

專案的所有自定義樣式均位於 `src/all-in-one.css`。該檔案整合了所有元件與頁面的樣式，方便統一管理與快速修改。
